Create MatrixText object, and set the function used to write to the display.

Parameters

set_xyPointer to function used to set a pixel on the display.  Should be of type “void (byte x, byte y, byte val)”.  Function set_xy will be called with val=0 to clear a pixel, non-zero otherwise.

Show scrolling text on the display.

Parameters

The first parameter is the message to display, the next four specify the box to draw the text within.

msgmessage text to display.  This is not copied, so must not be changed after calling this function.
ul_xUpper left x
ul_yUpper left y
br_xBottom right x
br_yBottom right y
scroll_textOptional.  If present and set to false, text will be static.  Text will be drawn to screen on first call to loop only, unless it’s force_redraw paramter is set.

set_scroll_speed

void set_scroll_speed(uint16_t speed)

Set how fast the text should scroll.

Parameters

speedhow long to wait (in ms) before advancing the text (default 50ms).  Setting to zero causes the text to advance every time loop is called.

set_character_spacing

void set_character_spacing(uint8_t pixels)

Set the spacing between each character.  Defaults to 1.

Parameters

pixelsSets the amount of space, in pixels, between each character.  Defaults to 1 if not set.

loop

void loop(bool force_redraw = false)

Call loop repeatedly to keep the text scrolling.  Will only update the display if it’s required based on the speed set using set_scroll_speed.

Parameters

force_redrawif true, always re-draw text, even if position hasn’t changed.
